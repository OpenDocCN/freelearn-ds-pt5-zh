External Data Science Resources

The book has covered a very wide range of materials, from Hadoop to NoSQL and machine learning to enterprise data science. The information has been exhaustive, but due to space limitations, we have not gone deep into any one particular area.

In the next few sections, we have provided links to external material for further reading that you may find useful should you want to learn more about the respective topics.

Several of the solutions here are either open source or community versions of enterprise software. The latter can be especially useful if you are considering enterprise solutions for big data and/or data science.
